# taxi-service
AF 2021 practice exam   
Technologies: KoaJS backend, MongoDB, and ReactJS frontend  
<h3>Installation</h3>
Within client and server directories
<pre>
$npm install
$npm start
</pre>
<h5>For jest tests</h5>
<pre>
$npm test
</pre>

startall.bat will execute *npm start* within both directories
